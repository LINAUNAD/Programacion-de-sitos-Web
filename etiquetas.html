ETIQUETAS HTML

Una etiqueta o marca HTML es un código que se incluye en los archivos creados con el lenguaje HTML para estructurar, añadir significado o formato al contenido a una página web.

ETIQUETA DESCRIPCIÓN
< p > Delimita el contenido de un párrafo de texto
< h1 ... h6 > Define los títulos de las secciones de mayor importancia de la página.
< blockquote > Se emplea para indicar que el texto que encierra es una cita textual de otro texto externo
< span > Se emplea para marcar cualquier elemento que no se puede marcar con las otras etiquetas definidas
< strong > Sirve para marcar los elementos importantes
< br/ > Fuerza al navegador a insertar una nueva línea, equivale a presionar la tecla ENTER (o Intro) en ese mismo punto.
< a > Se emplea para enlazar todo tipo de recursos.
name = "texto" - Permite nombrar al enlace para que se pueda acceder desde otros enlaces.
href = "url" - Indica la URL del recurso que se quiere enlazar
< link > Se emplea para enlazar y establecer relaciones entre el documento y otros recursos
< ol > Se emplea para definir listas ordenadas
< dl > Se emplea para definir listas de definición
< dt > Se emplea para definir los términos de los elementos de una lista de definición
< dd > Se emplea para indicar las definiciones de los elementos de una lista de definición
